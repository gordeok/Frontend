---

## 📁 폴더 구조

```text
GO르덕
├─ FE/        # 프론트엔드 
├─ BE/        # 백엔드
├─ AI/        # AI 
├─ docs/      # 문서 
```


---


## 🌿 Branch 규칙

> ✨ 기능 단위로 브랜치를 생성

### 📌 형식

```text
FE-기능명 / BE-기능명 / AI-기능명
```

### 📎 예시

* `FE-login`
* `BE-chat`

### 🚀 생성 방법

```bash
git checkout -b FE-login
git push -u origin FE-login
```


---


## 💬 Commit 규칙

> ✨ 커밋 메시지 아래 형식 참고

### 📌 형식

```text
타입: 작업 내용
```

### 📎 타입 종류

| 타입       | 설명       |
| -------- | -------- |
| feat     | 기능 추가    |
| fix      | 버그 수정    |
| design   | UI/UX 변경 |
| refactor | 코드 개선    |
| docs     | 문서 수정    |

### 📎 예시

* `feat: 로그인 기능 구현`
* `fix: 채팅 오류 수정`
* `design: 홈 화면 UI 수정`
